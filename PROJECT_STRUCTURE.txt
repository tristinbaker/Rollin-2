================================
Key Systems:
================================

GAME LOOP (game.py):
- 60 FPS with delta time (16.67ms per frame)
- 320x240 internal resolution, scaled 2x to 640x480
- Double buffering for smooth rendering

STATE MACHINE (game_state_manager.py):
- Manages menu, levels, pause states
- Tracks global state: score, lives
- Lazy state initialization

LEVEL SYSTEM (level_state.py):
- Base class for all levels
- Tiled map integration
- Entity spawning from map layers
- Parallax backgrounds
- HUD rendering
- Death/win screens

PLAYER (player.py):
- Physics: gravity, jumping, gliding
- HP system (3 hearts)
- Invincibility frames on hit
- Moving platform interaction
- Underwater physics support

ENEMIES:
- Slime: Patrols left/right between walls
- Bat: Flies in zigzag pattern (up-right, down-right)
- Wasp: Activates near player, flies left in sin wave (4 color variants)

TILEMAP (tilemap.py):
- Loads Tiled JSON (.tmj) format
- 32x32 pixel tiles
- 4-corner collision detection
- Entity position extraction from layers
- Camera offset rendering

MOVING PLATFORMS:
- Vertical and horizontal
- Defined in Tiled with properties (speed, distance)
- Player tracking (moves with platform)
- Enemies can stand on platforms

AUDIO (audio_manager.py):
- Background music with looping/fading
- Sound effects with volume control
- Pygame mixer integration

================================
Entity Spawning (Data-Driven):
================================

Entities spawn from Tiled map layers:
- Collision: Solid tiles for collision
- Coin: Coin spawn positions
- Spike: Spike hazard positions
- Slime: Ground enemy spawns
- Bat: Flying enemy spawns
- Wasp: Sin wave enemy spawns (tile ID = color)
- VerticalPlatform: Vertical moving platforms (properties: speed, distance)
- HorizontalPlatform: Horizontal moving platforms (properties: speed, distance)

================================
Physics Constants:
================================

Player Movement:
- move_speed = 0.3
- max_speed = 1.6
- stop_speed = 0.4

Gravity:
- fall_speed = 0.15
- max_fall_speed = 4.0

Jumping:
- jump_start = -4.8
- stop_jump_speed = 0.3

Gliding:
- glide_scale = 0.1 (90% fall speed reduction)

Underwater (optional):
- underwater_move_speed = 0.15
- underwater_max_speed = 0.8
- underwater_fall_speed = 0.05
- underwater_max_fall_speed = 1.5
