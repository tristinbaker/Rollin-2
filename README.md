# Rollin' 2

The sequel to a 2D platformer originally made as a Java college assignment about ten years ago, this time in Python, and expanded into a full game.

## What is it?

You run, jump, and glide your way through five levels collecting coins and dodging enemies. Each level has its own look, music, and set of hazards. Beat the game to unlock harder modes and if you're brave enough to 100% everything, there's a final challenge waiting.

## Controls

| Action | Keys |
|--------|------|
| Move | Arrow keys or WASD |
| Jump | Space |
| Glide | Hold Shift while falling |
| Pause | Escape |
| Confirm | Enter |

## Secrets

There's more to find beyond the first playthrough. Keep playing.
